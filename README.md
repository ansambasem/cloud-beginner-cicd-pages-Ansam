## Project Features
* **CI (Continuous Integration):** Automated checks using GitHub Actions to verify that all required files (index.html, style.css, script.js) exist before deployment.
* **CD (Continuous Deployment):** Automatic deployment to GitHub Pages every time a change is pushed to the main branch.
* **Interactive Element:** A JavaScript feature that responds to user interaction.

## How to run locally
1. Clone or download this repository.
2. Open the index.html file in any modern web browser.
